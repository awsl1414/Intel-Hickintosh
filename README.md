# Intel 13/14代 Hickintosh

## 配置详情

**CPU**：Intel I5 及以上 CPU
**GPU**：RX580/RX6600xt/RX6800xt (其他显卡自测)(xx50 请启用 SSDT-6x50_Spoof)  
**网卡/蓝牙**：Intel AX211  
**声卡**：瑞昱 ALC897  

## 细节

**OpenCore**：1.0.4


## 提示

请根据自己的主板找对应的分支
